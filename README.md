# jm-spy 

> jm-spy将[Jasmine 2.0](http://jasmine.github.io/2.0/introduction.html)中的spies封装成独立的模块，用于单元测试中对象方法的mock服务，以及跟踪记录被监测方法的调用及参数情况

## 示例：

```
$ npm install
$ make test TESTS=examples/spyOn.coffee
```
